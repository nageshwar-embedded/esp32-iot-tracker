# ESP32 IoT Secure Tracker

## 🚀 Overview
This project is a secure IoT-based tracking system built using ESP32, integrating biometric authentication, GPS tracking, environmental sensing, and a real-time web dashboard.

The system ensures that only authorized users can access tracking data using fingerprint authentication.

---

## ✨ Key Features
- 🔐 Fingerprint-based authentication system
- 📍 Real-time GPS tracking with map integration
- 🌡 Temperature and humidity monitoring (DHT11)
- 🌐 Web-based dashboard with live updates
- 🗺 Interactive map using Leaflet.js
- 📡 WiFi-based IoT system

---

## 🛠 Hardware Components
- ESP32
- GPS Module (NEO-6M)
- Fingerprint Sensor
- DHT11 Sensor

---

## 💻 Technologies Used
- Embedded C / Arduino
- HTTP Web Server
- TinyGPS++
- Adafruit Fingerprint Library
- JavaScript + HTML (UI)

---

## ⚙️ How It Works
1. User authenticates using fingerprint sensor  
2. ESP32 connects to WiFi  
3. GPS + sensor data is collected  
4. Data is displayed on a web dashboard  
5. Live map shows current location
